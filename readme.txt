== Installation ==

将Vaptcha-sms插件上传到您的博客，激活vaptcha-sms，然后在设置中配置Vid和Key。

e.g.

1. 上传vaptcha-sms插件到/wp-content/plugins/目录
2. 通过WordPress中的“插件”菜单激活插件
3. 登录VAPTCHA管理后台创建验证单元并开通短信
4. 登录插件后台完成配置并保存
